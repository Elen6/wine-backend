
# Project name
Wine-backend

## Description

This project is intended for an online store, ordering wines. 
It was built to help beginners who want to take the first  step in programming.



## Features

* MongoDB database
* CRUD
 

### Installing

git clone link
cd .. project
npm install


## Getting Started

nodemon server.js
