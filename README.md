
# Project name
Wine-backend

## Description

This project is intended for an online store, ordering and buying wines. 


## Features

* MongoDB database
* CRUD
 

### Installing

```
git clone https://github.com/Elen6/wine-backend.git
cd wine-backend
npm install 
npm run start

```

### Resourses

NodeJS
ExpressJS
Mongoose
MongoDB Compass
Nodemon
Dotenv
Cors
Nodmailer
